# CPAS 설계명세서 — 제8절: 항의 및 재시험

> **Cognitive Process Assessment System (CPAS)**
> 본 문서는 CPAS 설계명세서 제8절 "항의 및 재시험"을 독립 문서로 분기하여 상세화한 것이다.

---

## 8.1 재응시 왜곡 방어

재응시는 정당한 권리이지만, 재응시를 통해 부당한 이점을 얻는 것은 방어해야 한다. CPAS는 네 가지 방어 메커니즘을 적용한다.

### 8.1.1 구조 변이 (Structural Variant)

재응시 시 동일한 과제 인스턴스를 받지 않는다. 규칙 세트와 인지적 도전 구조는 동일하지만, 표면적 요소(변수값, 배치, 시각적 외형)가 무작위로 변경된 새로운 인스턴스를 받는다.

이 방어가 막는 것:
- 특정 과제의 "정답"을 암기하여 재응시에서 이점을 얻는 행위
- 타인으로부터 특정 인스턴스의 풀이 전략을 전수받는 행위

이 방어가 막지 못하는 것:
- 과제 유형 자체에 대한 경험적 학습 (마이크로월드 환경 자체에 익숙해지는 것)
- 과제의 인지적 구조에 대한 메타 이해 (어떤 유형의 정보가 깊이 숨겨져 있는지를 아는 것)

### 8.1.2 이상 탐지 (Anomaly Detection)

재응시 점수가 이전 점수와 통계적으로 비정상적인 차이를 보이는 경우를 자동으로 탐지한다.

탐지 기준:
- 점수 변화량이 규준 집단의 재검사 변동 분포에서 상위 5% 이상에 해당하는 경우
- 특정 차원의 점수만 급격히 변화한 경우 (예: 메타인지만 급상승)
- 프로세스 메트릭의 패턴이 이전 응시와 질적으로 완전히 다른 경우

이상 탐지 후 절차:
- 자동 플래그 설정
- 전문 심사관의 로그 비교 분석
- 필요 시 추가 인터뷰 실시
- 대리 응시, 외부 도구 사용 등의 부정행위 가능성 조사

### 8.1.3 엘리트 구분 (Elite Differentiation)

최상위 점수대(예: 상위 1%)에서의 재응시에 대해 추가적인 검증을 적용한다.

근거:
- 상위 1% 점수대에서의 미세한 점수 차이는 측정 오차의 영향을 크게 받음
- 극단적으로 높은 점수를 재현하는 것이 실력의 증거인지, 과제 구조에 대한 메타 이해의 결과인지를 구분해야 함
- 상위 점수대에서의 재응시 점수 상승이 실제 인지 능력 향상을 반영하는지 검증 필요

추가 검증:
- 이전 응시와 질적으로 다른 과제 소재 유형을 포함
- 인터뷰 심도를 높여 메타인지 정확도를 더 정밀하게 측정
- 이전 응시 로그와의 전략 패턴 비교 분석

### 8.1.4 인터뷰 필터 (Interview Filter)

재응시자에 대한 인터뷰에서 "이전 응시 경험이 현재 수행에 어떤 영향을 미쳤는가"를 탐색하는 추가 질문을 포함한다.

추가 질문 예시:
- "이전에 이 시스템을 경험한 적이 있습니까? 있다면, 이전 경험이 오늘 수행에 영향을 미쳤다고 생각합니까?"
- "오늘 과제에서 이전에 알고 있던 정보를 활용한 부분이 있습니까?"

이 질문의 목적은 부정행위 탐지가 아니라, 경험 효과의 크기를 추정하여 점수 보정에 반영하는 것이다.

## 8.2 항의 사유

응시자가 점수에 이의를 제기할 수 있는 정당한 사유를 명시한다.

### 8.2.1 인정되는 항의 사유

| 사유 | 설명 | 필요 증거 |
|------|------|-----------|
| 기술적 장애 | 마이크로월드, 입력 장치, 로깅 시스템의 기술적 오류로 수행이 방해받은 경우 | 시스템 로그에서의 오류 기록 |
| 환경 방해 | 시험장의 소음, 장비 고장 등 외부 요인으로 수행이 방해받은 경우 | 감독관 보고서, 시험장 기록 |
| 신체적 급성 이변 | 시험 중 급성 신체 이상(두통, 구토, 부상 등)이 발생한 경우 | 의료 기록 또는 감독관 보고서 |
| 입력 장치 보정 오류 | 캘리브레이션 과제에서의 보정 계수가 비정상적이었던 경우 | 캘리브레이션 로그 분석 |
| 인터뷰 절차 위반 | 인터뷰어가 구조화된 절차를 따르지 않은 경우 | 인터뷰 녹화/녹음 기록 |

### 8.2.2 인정되지 않는 항의 사유

| 사유 | 불인정 이유 |
|------|-------------|
| "평소 실력보다 못 봤다"는 주관적 판단 | 컨디션 변동은 모든 응시자에게 적용되는 조건이며, 사전 연기 제도가 제공됨 |
| "문제가 너무 어려웠다" | 과제 난이도는 구조 변이 범위 내에서 IRT로 보정됨 |
| "이 유형의 과제를 처음 접했다" | 과제 유형에 대한 사전 경험은 인지 능력의 일부이며, 모든 응시자가 동일하게 처음 접함 |
| "시험 시간이 부족했다" | 시간 제한은 과제 설계의 일부이며, 모든 응시자에게 동일하게 적용 |

## 8.3 재시험 절차

### 8.3.1 재시험 자격

| 조건 | 기준 |
|------|------|
| 최소 간격 | 마지막 응시로부터 6개월 이상 경과 |
| 항의 기반 재시험 | 항의 사유가 인정된 경우, 최소 간격 없이 재시험 가능 |
| 최대 횟수 | 연간 2회까지 (항의 기반 재시험 제외) |

### 8.3.2 재시험 과제 배정

- 이전 응시와 다른 구조 변이 인스턴스를 배정
- 가능한 한 이전 응시와 다른 과제 시퀀스 순서를 배정
- 이전 응시에서 기술적 장애가 발생한 과제 유형은 우선 재배정

### 8.3.3 점수 처리

| 상황 | 처리 |
|------|------|
| 일반 재시험 | 가장 최근 점수를 공식 점수로 사용. 이전 점수는 내부 기록에 보관되되 외부에 전달되지 않음 |
| 항의 기반 재시험 | 재시험 점수와 이전 점수 중 높은 점수를 공식 점수로 사용 |
| 이상 탐지 발동 시 | 심사 완료 전까지 점수 확정을 보류. 심사 결과에 따라 재시험 점수 인정 또는 추가 조사 |

## 8.4 사전 연기 제도

### 8.4.1 목적

컨디션 맹점(7.8절)의 영향을 완화하기 위한 제도. 응시자가 시험 당일 자신의 컨디션이 인지 수행에 체계적 영향을 미칠 수준이라고 판단하면, 시험을 연기할 수 있다.

### 8.4.2 운영 규칙

| 항목 | 기준 |
|------|------|
| 연기 가능 시점 | 시험 시작 전까지 (과제 배터리 시작 후에는 연기 불가) |
| 연기 횟수 제한 | 1회 접수당 1회 연기 가능 |
| 연기 시 재일정 | 7일~30일 이내로 재일정 |
| 연기 사유 신고 | 사유 신고 의무 없음 (자기 판단에 의한 연기) |
| 캘리브레이션 완료 후 연기 | 캘리브레이션 과제까지는 수행 후 연기 가능 (캘리브레이션 데이터는 보관) |

### 8.4.3 남용 방지

사전 연기 제도의 남용(반복적 연기로 시험 회피)을 방지하기 위해:

- 연기는 접수당 1회로 제한
- 연기 후 재일정에 다시 불참하면 해당 접수는 소멸되며 새로 접수해야 함
- 연기 이력은 내부 기록에 보관되되, 채점에 영향을 미치지 않음

### 8.4.4 시험 중 중단

오리엔테이션과 캘리브레이션을 마친 후 과제 배터리를 시작한 이후에는 연기가 불가능하다. 단, 급성 신체 이상이 발생한 경우 감독관의 판단으로 시험을 중단할 수 있으며, 이 경우 항의 기반 재시험의 자격이 자동으로 부여된다.

## 8.5 항의 처리 절차

### 8.5.1 제출

- 점수 통보 후 **30일 이내**에 서면으로 제출
- 항의 사유, 증거 자료, 요청 사항(재시험, 점수 재산출, 기타)을 명시

### 8.5.2 심사

- 항의 접수 후 **14일 이내**에 1차 심사 결과 통보
- 1차 심사는 시스템 로그, 감독관 보고서, 인터뷰 기록 등 객관적 증거에 기반
- 1차 심사에 불복하는 경우, **7일 이내**에 2차 심사(독립 심사 위원회) 요청 가능
- 2차 심사 결과는 최종이며, 추가 항소 불가

### 8.5.3 구제 조치

| 심사 결과 | 구제 조치 |
|-----------|-----------|
| 항의 인정 — 기술적 장애 | 재시험 기회 제공 (무료, 최소 간격 면제) |
| 항의 인정 — 환경 방해 | 재시험 기회 제공 (무료, 최소 간격 면제) |
| 항의 인정 — 보정 오류 | 보정 계수를 수정하여 점수 재산출 |
| 항의 인정 — 인터뷰 절차 위반 | 인터뷰 재실시 후 메타인지 차원 점수 재산출 |
| 항의 불인정 | 기존 점수 유지. 일반 재시험 절차를 통해 재응시 가능 |
