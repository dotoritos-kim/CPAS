# CPAS 설계명세서 — 제3절: 평가 항목

> **Cognitive Process Assessment System (CPAS)**
> 본 문서는 CPAS 설계명세서 제3절 "평가 항목"을 독립 문서로 분기하여 상세화한 것이다.

---

## 3.1 평가 항목의 이중 구조

CPAS의 평가 항목은 두 범주로 나뉜다:

| 범주 | 정의 | 역할 |
|------|------|------|
| 프로세스 메트릭 | 과제 수행 **과정**에서 추출하는 행동 패턴 지표 | 인지 구조의 직접 측정 |
| 결과 메트릭 | 과제 수행 **결과**의 질을 평가하는 지표 | 프로세스 메트릭의 외적 타당성 검증 |

CPAS의 고유성은 **프로세스 메트릭이 주(主) 채점 기준**이라는 점에 있다. 결과 메트릭은 보조적 역할을 하며, 프로세스 메트릭이 실제 인지 능력을 반영하는지를 검증하는 데 사용된다.

---

## 3.2 프로세스 메트릭 8종

1ms 틱 로그에서 추출하는 8개의 프로세스 메트릭이다. 각 메트릭은 9개 인지 차원(제1절) 중 하나 이상과 대응된다.

### 메트릭 1: 탐색 패턴 (Exploration Signature)

**대응 차원:** 발산적 사고, 사고의 깊이

정보 조회 순서와 빈도에서 추출하는 패턴. 응시자가 마이크로월드의 정보 공간을 어떻게 탐색하는지를 정량화한다.

세부 지표:
- **탐색 폭 (Breadth):** 전체 정보 노드 중 접근한 노드의 비율
- **탐색 순서 체계성 (Systematicity):** 탐색 순서가 무작위인지, 논리적 구조를 따르는지. 엔트로피 기반으로 정량화한다. 낮은 엔트로피 = 높은 체계성.
- **깊이 접근 빈도 (Depth Penetration):** "깊이 있게 생각하지 않으면 접근하지 않을 정보"에 접근한 횟수
- **중복 탐색률 (Revisit Rate):** 이미 조회한 정보를 다시 조회한 비율. 적정 수준의 중복은 확인 행동(verification)을 의미하고, 과도한 중복은 작업기억 한계를 시사한다.

### 메트릭 2: 전환 지연 (Transition Latency)

**대응 차원:** 사고의 깊이, 수렴적 사고

행동 A에서 행동 B로 넘어가는 데 걸린 시간. 두 행동 사이의 "빈 시간"은 내적 연산이 일어나고 있는 구간이다.

세부 지표:
- **평균 전환 지연:** 전체 행동 쌍의 평균 간격
- **전환 지연 분포:** 지연 시간의 분포 형태. 이봉(bimodal) 분포는 "빠른 자동 반응"과 "느린 숙고 반응"의 혼합을 시사한다.
- **맥락별 전환 지연:** 과제의 복잡도 구간별로 분리한 지연 시간. 복잡한 구간에서 지연이 증가하는 것은 정상이며, 복잡한 구간에서도 지연이 증가하지 않으면 얕은 처리를 시사한다.
- **물리적 조작 시간 분리 후 순수 인지 지연:** 운동 시간을 빼고 남은 순수 인지 처리 시간

### 메트릭 3: 되돌림 빈도 (Backtrack Rate)

**대응 차원:** 수렴적 사고, 메타인지

이전에 확인한 정보를 다시 확인하거나, 이전에 취한 행동을 번복하는 패턴.

세부 지표:
- **정보 재조회율:** 동일 정보 노드를 2회 이상 조회한 비율
- **행동 번복률:** 이전 행동을 취소하거나 반대 행동을 수행한 비율
- **전략적 되돌림 vs 혼란 되돌림:** 새로운 정보를 얻은 후 기존 정보를 재확인하는 것(전략적)과, 방향 없이 이미 본 것을 반복적으로 확인하는 것(혼란)을 구분. 전략적 되돌림 직전에 새로운 정보 접근이 있었는지로 판별한다.

### 메트릭 4: 행동 클러스터링 (Action Clustering)

**대응 차원:** 수렴적 사고, 분기 간 연관성

연속된 행동들이 하나의 **전략적 단위(strategic unit)**를 이루는지를 분석한다. 행동 간 간격이 짧고 목표 지향적인 연속 행동들은 하나의 클러스터로 묶인다.

세부 지표:
- **클러스터 크기 분포:** 전략적 단위의 평균 크기와 분포. 큰 클러스터 = 복잡한 계획의 연속 실행.
- **클러스터 간 간격:** 전략적 단위 사이의 "사고 구간" 길이. 이 구간은 다음 전략을 계획하는 시간으로 해석된다.
- **클러스터 내 일관성:** 클러스터 내 행동들이 하나의 목표를 향해 수렴하는지, 아니면 여러 목표를 혼합하는지
- **클러스터 전환 패턴:** 한 전략적 단위에서 다음 단위로의 전환이 논리적인지

### 메트릭 5: 환경 변화 후 반응 시간 (Post-Change Response Time)

**대응 차원:** 적응적 재계획

환경이 변화한 직후 응시자가 첫 행동을 취하기까지의 시간, 그리고 이후 행동 패턴의 변화.

세부 지표:
- **변화 탐지 지연:** 환경 변화가 발생한 시점과 응시자가 변화를 인식한 시점(변화 관련 정보를 조회한 시점) 사이의 간격
- **첫 적응 행동 지연:** 변화 인식 후 변화에 대응하는 첫 행동까지의 시간
- **계획 폐기 속도:** 변화로 인해 무효화된 기존 계획을 얼마나 빠르게 포기하는가
- **새 계획 수립 시간:** 변화 후 새로운 전략적 행동 클러스터가 형성되기까지의 시간
- **적응 후 수행 질:** 새 전략의 객관적 질이 변화 전 전략과 비교하여 어떠한가

### 메트릭 6: 발산-수렴 사이클 (Diverge-Converge Cycle)

**대응 차원:** 발산적 사고, 수렴적 사고

탐색의 폭이 넓어지는 구간(발산)과 특정 방향으로 좁혀지는 구간(수렴)의 교대 패턴.

세부 지표:
- **사이클 수:** 과제 수행 중 발산-수렴 사이클이 몇 회 반복되는가
- **사이클당 소요 시간:** 각 사이클의 평균 길이
- **발산-수렴 전환 타이밍:** 발산에서 수렴으로 전환되는 시점이 최적 시점과 얼마나 가까운가. 너무 빨리 수렴하면 충분히 탐색하지 않은 것이고, 너무 늦게 수렴하면 시간을 낭비한 것이다.
- **사이클 간 학습:** 이후 사이클에서 이전 사이클의 결과를 활용하는지

### 메트릭 7: 정보 통합 지표 (Information Integration Index)

**대응 차원:** 분기 간 연관성, 관점 전환

서로 다른 정보 원천에서 얻은 정보를 결합하여 새로운 행동을 도출하는 패턴.

세부 지표:
- **교차 참조 빈도:** 서로 다른 정보 영역의 노드를 연속적으로 조회하는 패턴
- **통합 행동 빈도:** 단일 정보 원천만으로는 도출할 수 없는 행동을 수행한 횟수
- **분기 전환 패턴:** 한 사고 분기에서 다른 분기로 전환할 때의 규칙성. 무작위 전환 vs 이전 분기의 결과에 기반한 전략적 전환.

### 메트릭 8: 메타인지 마커 (Metacognitive Markers)

**대응 차원:** 메타인지, 인지적 몰입 의지

과제 수행 중 자기 사고 과정을 모니터링하고 조절하는 행동의 흔적.

세부 지표:
- **자발적 점검 행동:** 과제가 요구하지 않았는데 자발적으로 이전 결과를 확인하는 행동
- **전략 전환의 자발성:** 외부 피드백 없이 전략을 전환한 빈도. 자발적 전환은 메타인지적 모니터링의 증거다.
- **난이도 적응 행동:** 과제가 어려워졌을 때 탐색 전략을 의식적으로 변경하는 패턴
- **포기 직전 행동:** 과제를 포기하기 직전의 행동 패턴. 체계적으로 시도한 후 포기하는 것(능력 한계)과 시도 없이 포기하는 것(의지 한계)을 구분한다.

## 3.3 계단 도달 측정 (Step-Reach Measurement)

### 3.3.1 개념

과제 난이도를 계단(step)으로 구조화하고, 응시자가 어느 계단까지 도달하는지를 측정한다. 각 계단은 이전 계단의 해결을 전제로 하며, 계단이 높아질수록 더 깊은 추론, 더 넓은 탐색, 더 정교한 전략이 필요하다.

### 3.3.2 계단의 설계 원칙

- 각 계단은 **명확한 인지적 난이도 증분**을 갖는다. "약간 더 어려운" 것이 아니라, "추가적인 인지 연산 X가 필요한" 것이다.
- 계단 간 전이는 객관적으로 정의 가능해야 한다. "이 계단에 도달했다"는 판정이 채점자의 주관에 의존하면 안 된다.
- 행동 로그에서 "이 계단에 도달했다"를 자동으로 판정할 수 있어야 한다.

### 3.3.3 계단 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| 추론 깊이 계단 | 추론 체인의 단계 수로 정의 | 2단계 추론 → 3단계 추론 → 4단계 추론 |
| 정보 통합 계단 | 통합해야 할 정보 원천의 수로 정의 | 2개 원천 통합 → 3개 원천 통합 |
| 동적 복잡도 계단 | 환경 변화의 빈도/규모로 정의 | 단일 변화 → 연쇄 변화 → 동시 다발 변화 |
| 자원 제약 계단 | 사용 가능한 자원의 감소로 정의 | 시간 충분 → 시간 보통 → 시간 부족 |

## 3.4 능력 천장과 의지 천장의 분리

### 3.4.1 핵심 개념

CPAS가 반드시 구분해야 하는 두 가지 한계:

- **능력 천장 (Ability Ceiling):** 응시자가 인지적으로 도달할 수 있는 최대 난이도. 더 높은 계단에 도달하려고 시도하지만 실패하는 지점.
- **의지 천장 (Will Ceiling):** 응시자가 인지적으로 도달할 수 있음에도 불구하고 시도를 중단하는 지점. 더 높은 계단에 도달할 능력이 있지만 시도하지 않는 지점.

### 3.4.2 분리 방법

두 천장을 분리하는 핵심 기준은 **포기 직전 행동 패턴**이다:

| 패턴 | 해석 | 천장 유형 |
|------|------|-----------|
| 높은 계단에서 다양한 전략을 시도한 후 실패 | 체계적 시도 후 한계 도달 | 능력 천장 |
| 높은 계단에 도달하자마자 탐색 없이 중단 | 시도 의지 부재 | 의지 천장 |
| 높은 계단에서 탐색을 시작했으나 빠르게 포기 | 부분적 시도 후 의지 소진 | 의지 천장 (부분적) |
| 낮은 계단에서 시간을 과도하게 소비하고 높은 계단에 도달하지 못함 | 완벽주의 또는 비효율적 탐색 | 맥락에 따라 판단 |

### 3.4.3 측정의 어려움

능력 천장과 의지 천장의 분리는 CPAS 설계에서 가장 미묘한 부분 중 하나다. 특히 다음 경우에 분리가 어렵다:

- 응시자가 능력 한계에 도달했지만 본인은 "시도하기 싫어서 그만뒀다"고 보고하는 경우 (자기보호적 메타인지 왜곡)
- 응시자가 의지가 없어서 그만뒀지만 본인은 "최선을 다했는데 못했다"고 보고하는 경우 (사회적 바람직성 편향)
- 인지적 피로로 인해 능력이 일시적으로 저하된 경우 (능력 천장의 동적 변동)

이런 경우 인터뷰 보정(제4절)과 로그-보고 일치 분석(제5절)이 개입하여 교정한다.

## 3.5 결과 메트릭

과제 수행의 결과물을 평가하는 지표. 프로세스 메트릭의 외적 타당성을 검증하는 역할을 한다.

### 3.5.1 목표 달성도

- 과제 내 목표를 얼마나 달성했는가 (0~100%)
- 부분 달성의 질: 50% 달성이라도 핵심 하위 목표를 달성한 경우와 부차적 하위 목표만 달성한 경우를 구분

### 3.5.2 최적성 비교

- 응시자의 행동 경로와 이론적 최적 경로의 차이
- 최적 경로 대비 추가 행동 수, 추가 시간, 추가 자원 소비량
- 차선 경로와의 비교: 최적은 아니더라도 합리적인 경로를 선택했는지

### 3.5.3 오류 패턴

- 오류의 총 횟수보다 **오류의 유형과 분포**가 더 중요하다
- 체계적 오류(특정 규칙을 잘못 이해)와 무작위 오류(주의 분산)의 구분
- 오류 후 교정 행동의 유무와 속도
- 같은 오류의 반복 여부 (학습 능력의 역지표)

### 3.5.4 결과 메트릭의 한계

결과 메트릭만으로는 CPAS의 목표를 달성할 수 없다. 그 이유:

- 결과가 좋더라도 과정이 비체계적일 수 있다 (운 좋게 맞은 경우)
- 결과가 나쁘더라도 과정이 체계적일 수 있다 (어려운 문제에 체계적으로 도전한 경우)
- 결과 메트릭은 능력 천장과 의지 천장을 구분하지 못한다

따라서 결과 메트릭은 프로세스 메트릭과 **교차 검증**하는 용도로만 사용하며, 종합 점수 산출에서의 가중치는 프로세스 메트릭보다 낮게 설정된다.

## 3.6 프로세스 메트릭과 인지 차원의 매핑

| 인지 차원 | 주요 프로세스 메트릭 | 보조 프로세스 메트릭 |
|-----------|---------------------|---------------------|
| 발산적 사고 | 탐색 패턴 | 발산-수렴 사이클 |
| 수렴적 사고 | 전환 지연, 행동 클러스터링 | 발산-수렴 사이클, 되돌림 빈도 |
| 사고의 깊이 | 탐색 패턴 (깊이 접근 빈도) | 전환 지연 (맥락별) |

| 적응적 재계획 | 환경 변화 후 반응 시간 | 되돌림 빈도 (전략적 되돌림) |
| 메타인지 | 메타인지 마커 | 되돌림 빈도, 발산-수렴 사이클 |
| 관점 전환 | 정보 통합 지표 | 탐색 패턴 (역할 전환 시 변화) |
| 비판적 사고 | 탐색 패턴 (함정 정보 관련) | 보완 과제 전용 메트릭 (9절) |
| 인지적 몰입 의지 | 메타인지 마커 (포기 패턴) | 탐색 패턴 (난이도별 탐색 지속 시간) |
